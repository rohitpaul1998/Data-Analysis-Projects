NYC food inspection analysis

1. Dataset gathered from https://data.cityofnewyork.us/Health/DOHMH-New-York-City-Restaurant-Inspection-Results/43nn-pn8j

2. Designed data preparation and data profiling workflow pipelines to Extract Transform and Load raw data into staging table in MS SQL Server

3. Designed a dimensional data model to identify dimensions, facts and key relationships

4. Loaded the integration schema in SQL server with cleansed and transformed data using Alteryx 

5. Performed data visualizations in Power BI and Tableau to extract key insights to drive business and make better decision making strategies
